ASO114开发者沙龙广州站总结 20171027

发表于 2017-10-27 | 分类于 营销推广

参加了ASO114全国开发者沙龙第二期，广州站的分享，收获了很多很有价值的内容，这里做下总结分享。

分享的内容分为三部分，下面分点总结。

一、ASO3.0时代，开发者应该如何应对

  1. 精耕细作，精细化运营，不放过任何流量；
  2. 苹果份额逐渐在缩小，国产必然崛起，应用宝为首，华米OV随后成为第一梯队；

二、APP推广爆发式增长的黑科技（技巧）

有别于传统刷量的更有效的刷量方式——爆量

  1. 最短的时间完成最大的下载量级
  2. 真实流量洗白
  3. 规避爆量风险

解读：
新产品上架，在前2-3天刷个几百万的量，后面就不用再刷了。相对的，APP更新版本，可以在审核周期内把量爆完（利用周末）。

一般不会被识别到刷量，假如被识别到了被警告，一般在警告所提示的宽限时间内爆完，也不会被惩罚。假如真的被惩罚了，可以通过更新版本来解决。假如连更新都不给，就只能去卖嘴皮子求情了（比如说是被竞争对手恶意刷量，自己并不知情）。

假如产品这么操作被警告多次了，就不适合再继续操作下去了。

有别于传统刷评论的更有效的刷评论方式——爆评（针对苹果）

1、一定真实量／广告量铺垫基础权重
2、集中爆评
3、规避风险

解读：
只要没做过机刷的APP都可以做爆评。操作原理类似爆量，即是在最短的时间完成最大的评论量级。在苹果整治前爆评，不适合通过积分墙爆评，适合机刷。刷评论，机刷是可以补量的，积分墙不可以

提升积分墙效果

  1. 排重。过滤已安装过APP的用户重复安装应用；过滤多家渠道之间的重复用户。
  2. 点击上报。主动获取用户新增数据，第一时间了解任务的完成情况以及进度。
  3. 激活上报。以收到真实激活数据作为结算考核，防止渠道掺假量和假数据。
  4. 回调。调整数据回调逻辑，提升积分墙的打量效果（所谓回调，即是可以要求积分墙任务为，隔天、后天再打开多少分钟之类）。

抓住爆发式增长的点

举例：

  1. 邀请激励机制（ofo、滴滴）
  2. UGC模式（Faceu、快手）
  3. 诱导分享（王者荣耀）
  4. 满足社交心理（Keep、全民K歌）

三、世界500强企业的实操经验分享

500强企业推APP的优劣势

优势

  1. 有钱不傻，预算充足
  2. 品牌知名度高
  3. 上架时间久
  4. 重视前端下载评论，差评零容忍
  5. 权重高

劣势

  1. 只是单纯的更新版本
  2. 版本更新次数少
  3. 广告投放只知道竞价

关于前端下载评论

  1. 最简单直接提高权重的方法
  2. 你的竞争对手都在做
  3. 影响关键词优化效果

怎么做马甲包

  1. 主包不要和马甲包用一个开发着账号，且不要使用同一个IP，惩罚会容易被一锅端；
  2. 市面常见做法是弄公司壳来发马甲包；
  3. 马甲包在功能、UI、icon上不要太相似，尽量当成一款独立APP；
  4. 以量取胜，一个关键词做多个马甲包，农村包围城市。

关于权重

不同阶段影响权重最主要的因素不同：

  • 2014年：更新版本、下载评论
  • 2015年：搜索下载权重高
  • 2016年：新应用有扶持
  • 2017年：上架时间越早，权重越高

结合中小APP实操，应该注重的要点

  1. 前端下载量和评论
  2. 马甲包
  3. 更新频率
  4. 积分墙冲词
  5. APP标题描述一句话描述

现阶段所有玩法，速度要快，因为今天不知道明天应用市场会出什么新政策。app上线越早，权重越高，新的app无需等各方面完善再上线，可先上架个初版本优先占领市场，前期在用户少的情况下可多次更新，让功能各方面趋于完善，后期用户多了需稳定更新，一个月两次为宜。

四、Q＆A环节

Q：苹果的开发者账号有等级之分么，等级会影响权重么？
A：目前还是个推测性的结论，苹果现在正在构建一个开发者等级体系，而等级肯定会影响APP权重的。

Q：我们的产品通过后台添加关键词并刷量的方式，怎样都没办法覆盖到某个关键词，请问这种得怎么办？
A：安卓的关键词添加，除了优化一句话简介，副标题和简介外，小米和华为后台可添加100字符的关键词，而vivo是不能直接添加副标题，需通过技术手段单独打包。另外除了加副标题外，还要懂得刷词，比如要覆盖淘宝这个关键词，可拆成淘**宝，可以增加覆盖的几率。

Q：请问应用市场审核评论的规则是什么，为什么会出现真人评论被删，而机刷的却通过了的情况？
A：这个暂时没有什么规律可循。像苹果、华为这种审核比较严格的市场，都是人工审核，受人为影响太大，像我们自己测试华为评论，试过人为评论的基本被删，而机刷的通过了30%。

Q：我们的APP打算走出海外市场，请问这种应该选择在原有包增加英文版本还是直接弄一个新的包？
A：建议弄一个新的包，因为海外市场你要用的关键词必定是英文不说，关键词、包得内容肯定也是不是完全一样的，在海外用原来的包去做优化时不方便，且会影响国内的。

Q：金融类产品因为1.2原因审核被拒绝怎么办？
A：金融贷款类的产品，如因为1.2的原因被拒，可将其改成非金融属性产品，如修改成记账类，或资讯类，等审核通过再做热更新。资质问题或者可直接利用其他金融公司的壳上架，后面再修改回来，不会存在风险。或者也可以通过第三方金融公司授权的方式上架。另外，没有资质的金融产品最好少更新

Q:为什么我们的新产品在苹果刚上架时那段时间排名的情况都做得挺不错的，但是在后面就突然下降了？
A：苹果对新上架APP的加权期为7天，7天后会下降。

Q:如何鉴别真机刷和假机刷？
A:现在市面上的机刷基本都为真机刷，即是通过群控软件刷。很久以前有过协议刷，但这种方式基本都快消失了。

Q：我们的马甲包因为怕被抓到批量下架，所以自己先下架了，请问这样恢复上架会影响权重么？
A：会的。
